## Vim Basics

### Modes
- `i` → Insert mode (start typing)
- `Esc` → Exit insert mode and go to normal mode

### Editing
- `dd` → Delete current line
- `10dd` → Delete 10 lines
- `u` → Undo last change
- 'dG' - deletes from cursor to the end 

### Navigation
- `0` → Go to start of line
- `$` → Go to end of line

### Search & Replace
- `/pattern` → Search for a word or pattern
- `:%s/old/new` → Replace all occurrences of "old" with "new" in the file

### Save & Exit
- `:wq` → Save and quit Vim
